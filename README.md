MHTCETLAW

MHT CET Law Course Portal is a full-stack web application built using HTML, CSS, and JavaScript, integrated with Google Firebase Authentication and Cloud Firestore for secure user management and real-time data storage.

The platform is deployed on Netlify and provides:

Structured course modules

Organized study materials

Mock tests and practice resources

Secure content access control

Authentication-based user dashboard

🚀 Tech Stack

HTML5

CSS3

JavaScript (Vanilla JS)

Google Firebase Authentication

Cloud Firestore

Netlify (Deployment)

📌 Features

Secure Login & Signup System

Role-based Content Access

Real-time Database Integration

Responsive UI Design

Protected Course Materials
